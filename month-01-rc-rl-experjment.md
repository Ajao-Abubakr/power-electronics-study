What You Document
This goes on GitHub as your first real project. Create a folder called projects in your repository. Inside it create month-01-rc-rl-explorer.md.
The document must contain:
1. Introduction
What is the purpose of this project. What theory does it demonstrate. Why RC circuits matter in power electronics.
2. Circuit Diagrams
Draw each circuit clearly. Label all component values.
3. Calculations
For each circuit: calculate τ, calculate expected voltage at each time point using v(t) = V_s(1 − e^(−t/τ)).
4. Results Table
Your measured values vs theoretical values for all three circuits.
5. Simulation Screenshots
LTspice waveforms for all three circuits.
6. Discussion
Do measurements match theory? If not, why not?
Which circuit charged fastest? Why?
What would happen if you doubled R in Circuit 1?
What would happen if you doubled C in Circuit 1?
7. Power Electronics Connection
Explain how this experiment connects to gate drive circuits and soft-start circuits in converters. Use the numbers from Topic 8.
